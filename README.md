# Secure Login System with SHA-256 Password Hashing

## Project Overview

This Python project demonstrates a secure login system that stores user passwords as SHA-256 hashes instead of plain text.

## Features

- User Registration
- User Login Authentication
- SHA-256 Password Hashing
- JSON File Storage
- Password Validation
- Secure Password Verification

## Technologies Used

- Python 3
- hashlib
- json
- os

## Security Concepts Demonstrated

- Password Hashing
- Authentication
- User Management
- Secure Credential Storage
